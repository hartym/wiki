These snippets use `analytics.js` (the `ga()` API) from Universal Analytics, which stopped
processing data on July 1, 2023. Kept for the record; Google Analytics 4 uses `gtag()` calls
instead.

```javascript
ga('send', 'pageview', 'name of your virtual page');
ga('send', 'event', 'event category', 'event action', 'event label', event value);
ga('send', 'event', 'event category', 'event action', 'event label', event value, {'nonInteraction': 1});
```
