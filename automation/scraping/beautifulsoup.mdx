## Recipes, Tips and Tricks

### How to remove comments from a bs4 element?

```python
from bs4 import Comment

for comment in soup.find_all(string=lambda text: isinstance(text, Comment)):
    comment.extract()
```

The camelCase `findAll` and the `text=` argument still work but are deprecated (since 4.0);
`find_all` and `string=` are the current spelling.

### How to remove all the AngularJS bullshit attributes so you can actually read the HTML?

```python
for tag in soup.descendants:
    if hasattr(tag, 'attrs'):
        tag.attrs = {
            key: value
            for key, value in tag.attrs.items()
            if not key.startswith('ng-')
        }
```

`.descendants` replaces the deprecated `recursiveChildGenerator()`. Note: this only removes the
ng-* attributes, not the ng-* classes.

Verified on beautifulsoup4 4.15.0 with the lxml parser.
