## Construct a DataFrame from a list of dicts

```python
import pandas as pd

ds = []
for user in users:
        ds.append({
            'screen_name': user.screen_name,
            'name': user.name
        })

    df = pd.DataFrame(ds)
```

## Create an index afterward

```python
df = df.set_index('screen_name')
```
