## Time Series Oriented Databases (TSDB)

### InfluxDB

[InfluxDB](https://github.com/influxdata/influxdb) - Scalable datastore for metrics, events, and real-time analytics

### TimescaleDB

[TimescaleDB](https://github.com/timescale/timescaledb) - Time-series database built on PostgreSQL

Timescale DB claims to be a horizontally scalable TSDB bundled as a PostgreSQL extension, allowing to _not_ run another database for timestamped data.

### OpenTSDB

[OpenTSDB](https://github.com/OpenTSDB/opentsdb) - Distributed, scalable Time Series Database on top of HBase

### Graphite

[Graphite](https://github.com/graphite-project/graphite-web) - Scalable real-time graphing system

* [Whisper](https://github.com/graphite-project/whisper) - Fixed-size database for storing time-series data
* [Carbon](https://github.com/graphite-project/carbon) - Backend data caching and persistence daemon

### RRDtool

[RRDtool](https://github.com/oetiker/rrdtool-1.x) - Round Robin Database for time series data

### ELK Stack

* [Elasticsearch](https://github.com/elastic/elasticsearch) - Distributed, RESTful search and analytics engine
* [Logstash](https://github.com/elastic/logstash) - Server-side data processing pipeline
* [Kibana](https://github.com/elastic/kibana) - Analytics and visualization platform

Alternatives:

* [Solr](https://github.com/apache/solr) - Enterprise search platform
* <Link to="/bi/silk" /> - SOLR + Logstash + Banana (Kibana fork)

### Other Time Series Databases

* [Prometheus](https://github.com/prometheus/prometheus) - Monitoring system and time series database
* [KairosDB](https://github.com/kairosdb/kairosdb) - Time series database on top of Cassandra
* [Druid](https://github.com/apache/druid) - Real-time analytics database
* [VictoriaMetrics](https://github.com/VictoriaMetrics/VictoriaMetrics) - Fast, cost-effective time series database
* [QuestDB](https://github.com/questdb/questdb) - High-performance time series database

## Geographic Information Systems

* [PostGIS](https://github.com/postgis/postgis) - Spatial database extender for PostgreSQL
* [MongoDB](https://github.com/mongodb/mongo) - Document database with geospatial queries support

### Tiles & Web

* [Awesome Vector Tiles](https://github.com/mapbox/awesome-vector-tiles) - Curated list of vector tiles resources
* [GeoDjango](https://github.com/django/django) - Django's geographic web framework
* [TileLive](https://github.com/mapbox/tilelive) - Map tile server and utilities
* [Tessela](https://github.com/urbica/tessella) - Lightweight Node.js Mapbox Vector Tiles server
* [Tippecanoe](https://github.com/mapbox/tippecanoe) - Build vector tilesets from large GeoJSON features

### Coordinates

* [EPSG.io](http://epsg.io/) - Coordinate systems worldwide
* [PyProj](https://github.com/pyproj4/pyproj) - Python interface to PROJ (cartographic projections library)

### Maps

* [Folium](https://github.com/python-visualization/folium) - Python data to Leaflet.js maps (works in Jupyter)

### Miscellaneous

* WGS 84 (World Geodetic System 1984): the standard default ([EPSG:4326](https://epsg.io/4326)).
* Lambert II: a French coordinate system ([EPSG:27572](https://epsg.io/27572)).

Lots of resources: [GovHack Toolkit - Geographic Data](http://govhack-toolkit.readthedocs.io/technical/geographic-data/)

[GeoFree - Coordinate Converter](http://geofree.fr/gf/coordinateconv.asp)

## Key-Value Stores

* [LevelDB](https://github.com/google/leveldb) - Fast key-value storage library by Google
* [Redis](https://github.com/redis/redis) - In-memory data structure store, used as database, cache, and message broker
* [MemcacheDB](https://github.com/stvchu/memcachedb) - Distributed key-value storage system designed for persistence
* [Couchbase](https://github.com/couchbase/couchbase-server) - Distributed NoSQL cloud database

## Document-Oriented Databases

* [Couchbase](https://github.com/couchbase/couchbase-server) - Distributed NoSQL cloud database
* [CouchDB](https://github.com/apache/couchdb) - Seamless multi-master sync, scales from mobile to global clusters
* [Elasticsearch](https://github.com/elastic/elasticsearch) - Distributed, RESTful search and analytics engine
* [MongoDB](https://github.com/mongodb/mongo) - The most popular NoSQL database
* [RethinkDB](https://github.com/rethinkdb/rethinkdb) - The open-source database for the realtime web
* [Solr](https://github.com/apache/solr) - Blazing-fast, open source enterprise search platform
