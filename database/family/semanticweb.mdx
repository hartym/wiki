## Standards

* [Linked Data Platform 1.0 - W3C](https://www.w3.org/TR/ldp/)

RDF uses subject–predicate–object triples.

## Python tools & guides

* [RDFLib](https://github.com/RDFLib/rdflib) - Python library for working with RDF
* [RDFLib-SQLAlchemy](https://github.com/RDFLib/rdflib-sqlalchemy) - SQLAlchemy store for RDFLib
* [SPARQLWrapper](https://github.com/RDFLib/sparqlwrapper) - SPARQL endpoint interface for Python
* [Survey of Python RDF tools](http://www.michelepasin.org/blog/2011/02/24/survey-of-pythonic-tools-for-rdf-and-linked-data-programming/) - Comprehensive overview

## Databases

* [Virtuoso](https://github.com/openlink/virtuoso-opensource) - High-performance object-relational SQL database
* [Apache Jena](https://github.com/apache/jena) - Java framework for building Semantic Web applications
* [Blazegraph](https://github.com/blazegraph/database) - High-performance graph database for RDF
* [GraphDB](https://github.com/Ontotext-AD/graphdb-free) - Enterprise triplestore with RDF and SPARQL support
